## How to run

### Backend
1. cd backend
2. copy `.env.example` to `.env` and set `MONGO_CONN` to your MongoDB Atlas connection string
3. npm install
4. npm run dev

### Frontend
1. cd frontend
2. copy `.env.example` to `.env` and set `REACT_APP_API_BASE=http://localhost:5600/api`
3. npm install
4. npm start
